# Python 삭제 가이드 (Windows)

## 1. 제어판에서 삭제

1. `Win + R` → `appwiz.cpl` 입력 후 Enter
2. 프로그램 목록에서 **Python 3.x.x** 찾기
3. 우클릭 → **제거(Uninstall)** 클릭
4. 언인스톨러 지시에 따라 완료

---

## 2. 설정 앱에서 삭제

1. `Win + I` → **앱** → **설치된 앱**
2. 검색창에 `Python` 입력
3. 해당 버전 클릭 → **제거** 선택

---

## 3. 잔여 파일 수동 삭제

삭제 후에도 아래 경로에 파일이 남을 수 있으므로 확인 후 삭제:

```
# 설치 폴더
C:\Users\<사용자명>\AppData\Local\Programs\Python\

# pip 캐시
C:\Users\<사용자명>\AppData\Local\pip\

# 사용자 스크립트
C:\Users\<사용자명>\AppData\Roaming\Python\
```

---

## 4. 환경 변수(PATH) 정리

Python이 PATH에 남아 있으면 오류가 발생할 수 있습니다.

1. `Win + R` → `sysdm.cpl` → **고급** 탭 → **환경 변수**
2. **시스템 변수** 또는 **사용자 변수** 의 `Path` 선택 → **편집**
3. Python 관련 경로 삭제:
   ```
   C:\Users\<사용자명>\AppData\Local\Programs\Python\Python3xx\
   C:\Users\<사용자명>\AppData\Local\Programs\Python\Python3xx\Scripts\
   ```
4. **확인** 클릭 후 터미널 재시작

---

## 5. 삭제 확인

```powershell
python --version
# 'python'은(는) 내부 또는 외부 명령... (정상 삭제 확인)
```

---

## 6. 주의사항

- 가상환경(venv)이 남아 있다면 해당 폴더도 별도로 삭제
- 여러 버전이 설치된 경우 각 버전마다 위 과정 반복
