# Python 설치 가이드 (Windows)

## 1. 공식 사이트에서 다운로드

1. [https://www.python.org/downloads/](https://www.python.org/downloads/) 접속
2. **Download Python 3.x.x** 버튼 클릭 (최신 안정 버전 권장)

---

## 2. 설치 진행

1. 다운로드된 `.exe` 파일 실행
2. **반드시** 아래 옵션 체크 후 설치 진행

   ```
   ☑ Add python.exe to PATH
   ```

3. **Install Now** 클릭 (기본 설치) 또는 **Customize installation** 선택

### Customize installation 권장 옵션

| 옵션 | 설명 |
|------|------|
| pip | 패키지 관리자 (필수) |
| tcl/tk and IDLE | 기본 IDE |
| Python test suite | 테스트 도구 |
| Add Python to environment variables | PATH 등록 (필수) |

---

## 3. 설치 확인

터미널(PowerShell 또는 CMD)을 **새로 열고** 아래 명령 실행:

```powershell
python --version
# Python 3.x.x

pip --version
# pip x.x.x from ...
```

---

## 4. 설치 경로 (기본값)

```
C:\Users\<사용자명>\AppData\Local\Programs\Python\Python3xx\
```

---

## 5. 주의사항

- 설치 시 `Add python.exe to PATH` 미체크 시 명령어 인식 불가
- 여러 버전 설치 시 `py -3.11`, `py -3.12` 형식으로 버전 지정 가능
- 기존 Python이 설치된 경우 버전 충돌에 주의
