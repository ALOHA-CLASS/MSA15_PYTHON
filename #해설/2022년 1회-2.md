# 2022년 1회 - 리스트 메서드 (extend / pop / reverse)

## 1. `extend()` — 요소 확장 (모든 항목을 개별 요소로 추가)

리스트에 다른 반복 가능한(iterable) 객체의 **모든 항목을 개별적으로** 추가합니다.  
`append()`는 통째로 하나의 요소로 넣지만, `extend()`는 **펼쳐서** 넣습니다.

```python
a = [1, 2, 3]
b = [4, 5, 6]

a.extend(b)
print(a)  # [1, 2, 3, 4, 5, 6]

# append와 비교
c = [1, 2, 3]
c.append(b)
print(c)  # [1, 2, 3, [4, 5, 6]]  ← 리스트 자체가 하나의 요소로 들어감
```

---

## 2. `pop()` — 요소 꺼내기 (삭제 후 반환)

리스트에서 특정 인덱스의 요소를 **삭제하면서 그 값을 반환**합니다.  
인덱스를 생략하면 **마지막 요소**를 꺼냅니다.

```python
a = [10, 20, 30, 40, 50]

val = a.pop()       # 마지막 요소 꺼내기
print(val)          # 50
print(a)            # [10, 20, 30, 40]

val2 = a.pop(1)     # 인덱스 1 요소 꺼내기
print(val2)         # 20
print(a)            # [10, 30, 40]
```

---

## 3. `reverse()` — 순서 뒤집기

리스트 내부 요소의 순서를 **제자리에서(in-place) 뒤집습니다.**  
새 리스트를 반환하지 않고 **원본 리스트를 직접 변경**합니다.

```python
a = [1, 2, 3, 4, 5]

a.reverse()
print(a)  # [5, 4, 3, 2, 1]

# 문자열 리스트도 가능
b = ['apple', 'banana', 'cherry']
b.reverse()
print(b)  # ['cherry', 'banana', 'apple']
```

---

## 요약 비교

| 메서드 | 역할 | 반환값 |
|--------|------|--------|
| `extend(iterable)` | 다른 리스트의 요소를 개별적으로 추가 | `None` |
| `pop(index)` | 해당 인덱스 요소를 삭제하고 반환 | 삭제된 요소 값 |
| `reverse()` | 리스트 순서를 뒤집음 (원본 변경) | `None` |
