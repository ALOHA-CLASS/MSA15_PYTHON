# 2021년 1회 - 클래스 변수 & 문자열 인덱싱

## 원본 코드

```python
class good:
  li = ["seoul", "kyeonggi","inchon","daejeon","daegu","pusan"]

g = good()
str01 = ''
for i in g.li:
  str01 = str01 + i[0]
  
print(str01)
```

---

## 문법 해설

### 클래스 변수 (Class Variable)

```python
class good:
  li = ["seoul", "kyeonggi","inchon","daejeon","daegu","pusan"]
```

- `li`는 클래스 내부에 선언된 **클래스 변수** — 모든 인스턴스가 공유
- 인스턴스 변수(`self.변수`)와 달리 `__init__` 없이 선언 가능

---

### 인스턴스 생성 및 클래스 변수 접근

```python
g = good()    # good 클래스의 인스턴스 생성
g.li          # 인스턴스를 통해 클래스 변수 접근 (good.li 와 동일)
```

---

### 문자열 인덱싱 `i[0]`

```python
for i in g.li:
    str01 = str01 + i[0]  # 각 문자열의 첫 번째 문자(인덱스 0) 추출
```

- 문자열도 인덱싱 가능: `"seoul"[0]` → `'s'`

---

## 실행 흐름

| i | i[0] | str01 누적 |
|---|------|------------|
| "seoul" | 's' | "s" |
| "kyeonggi" | 'k' | "sk" |
| "inchon" | 'i' | "ski" |
| "daejeon" | 'd' | "skid" |
| "daegu" | 'd' | "skidd" |
| "pusan" | 'p' | "skiddp" |

---

## 출력 결과

```
skiddp
```
