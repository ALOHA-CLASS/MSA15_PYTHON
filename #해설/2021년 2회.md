# 2021년 2회 - 비트 오른쪽 시프트 연산자 (`>>`)

## 원본 코드

```python
a = 100
result = 0
for i in range(1,3):
  result = a >> i
  result = result + 1
print(result)
```

---

## 문법 해설

### 비트 오른쪽 시프트 `>>`

```python
a >> i
```

- 정수를 2진수로 표현했을 때 비트를 **오른쪽으로 i칸 이동**
- 수학적으로: `a >> i` = `a // 2^i` (정수 나눗셈)

```
100의 2진수: 01100100
100 >> 1  = 00110010 = 50   (100 // 2 = 50)
100 >> 2  = 00011001 = 25   (100 // 4 = 25)
```

---

### `range(1, 3)` — 반복 범위

```python
for i in range(1, 3):   # i = 1, 2
```

- `range(시작, 끝)` : 시작 이상, 끝 **미만**
- `range(1, 3)` → 1, 2 (3 미포함)

---

## 실행 흐름

| i | `a >> i` | `result + 1` |
|---|----------|--------------|
| 1 | `100 >> 1` = 50 | 51 |
| 2 | `100 >> 2` = 25 | 26 |

> `result = a >> i` 이므로 매 반복마다 `a(100)`에서 시프트 — 이전 result를 누적하지 않음

---

## 출력 결과

```
26
```
