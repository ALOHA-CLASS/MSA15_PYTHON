## 원본 코드

def func(lst):

func(lst)

---

## 문법 해설

### 함수: 리스트 in-place 역순 정렬

```python
def func(lst):
  for i in range(len(lst) // 2):
    lst[i], lst[-i-1] = lst[-i-1], lst[i]
```

- `len(lst) // 2` = `6 // 2` = 3 → 앞뒤로 3쌍 교환
- `lst[-i-1]` : 음수 인덱스 — 뒤에서부터 접근
  - `lst[-1]` = 마지막 원소, `lst[-2]` = 뒤에서 두 번째 ...

| 음수 인덱스 | 의미 |
|-------------|------|
| `lst[-1]` | `lst[5]` (마지막) |
| `lst[-2]` | `lst[4]` |
| `lst[-3]` | `lst[3]` |

---

### 동시 교환 (Swap)

```python
lst[i], lst[-i-1] = lst[-i-1], lst[i]
```

- Python은 우변을 먼저 평가 후 좌변에 동시 할당 → 임시 변수 없이 스왑 가능

---

### func 실행 흐름 (lst = [1,2,3,4,5,6])

| i | lst[i] ↔ lst[-i-1] | lst 상태 |
|---|---------------------|----------|
| 0 | lst[0] ↔ lst[-1] = lst[5] | [**6**,2,3,4,5,**1**] |
| 1 | lst[1] ↔ lst[-2] = lst[4] | [6,**5**,3,4,**2**,1] |
| 2 | lst[2] ↔ lst[-3] = lst[3] | [6,5,**4**,**3**,2,1] |

→ 결과: `[6, 5, 4, 3, 2, 1]`

---

### 스텝 슬라이싱 `[::2]`, `[1::2]`

```python
lst[::2]    # 인덱스 0, 2, 4 → [6, 4, 2]  (짝수 인덱스)
lst[1::2]   # 인덱스 1, 3, 5 → [5, 3, 1]  (홀수 인덱스)
```

- `[start:end:step]` : start부터 end까지 step 간격으로 추출
- `[::2]` = 처음부터 끝까지 2칸 간격

---

### `sum()` 내장 함수

```python
sum([6, 4, 2])  # 12
sum([5, 3, 1])  # 9
```

= sum([6,4,2]) - sum([5,3,1])
