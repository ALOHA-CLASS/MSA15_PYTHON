# 2021년 3회 - 비교 연산자 (`==`)

## 원본 코드

```python
a, b = 100, 200
print(a == b)
```

---

## 문법 해설

### 다중 할당 (Multiple Assignment)

```python
a, b = 100, 200
```

- 한 줄에 여러 변수를 동시에 할당
- `a = 100`, `b = 200` 과 동일

---

### 비교 연산자 `==`

```python
a == b   # 100 == 200
```

- 두 값이 같으면 `True`, 다르면 `False` 반환
- 결과는 `bool` 타입

| 연산자 | 의미 |
|--------|------|
| `==` | 같다 |
| `!=` | 다르다 |
| `>` / `<` | 크다 / 작다 |
| `>=` / `<=` | 크거나 같다 / 작거나 같다 |

---

### `=` vs `==` 구분

| 기호 | 역할 |
|------|------|
| `=` | **대입** 연산자 (변수에 값 저장) |
| `==` | **비교** 연산자 (같은지 비교, bool 반환) |

---

## 출력 결과

```
False
```

> 100 ≠ 200 이므로 `False`
