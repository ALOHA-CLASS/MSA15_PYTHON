# 2024년 1회 - 리스트 순회 & 문자열 인덱싱

## 원본 코드

```python
a = ["Seoul", "Kyeonggi", "Incheon", "Daejun", "Daegu", "Pusan"]
str = "S"

for i in a:
  str = str + i[1]

print(str)
```

---

## 문법 해설

### 리스트 순회 `for ... in`

```python
for i in a:
```

- 리스트 `a`의 원소를 하나씩 `i`에 할당하며 반복
- `i`는 순서대로 각 도시 이름 문자열

---

### 문자열 인덱싱 `i[1]`

```python
str = str + i[1]   # 각 도시명의 인덱스 1번(두 번째 글자) 추출
```

- 문자열도 리스트처럼 인덱싱 가능
- `i[1]` → 각 단어의 **두 번째 문자**

---

### ⚠️ `str` 변수명 주의

```python
str = "S"
```

- `str`은 Python **내장 함수** (문자열 타입 변환)
- 변수명으로 사용하면 내장 함수가 **가려짐** (shadow)
- 실무에서는 `result`, `s` 등 다른 이름 권장

---

## 실행 흐름

| i | i[1] | str 누적 |
|---|------|---------|
| 초기 | | `"S"` |
| `"Seoul"` | `'e'` | `"Se"` |
| `"Kyeonggi"` | `'y'` | `"Sey"` |
| `"Incheon"` | `'n'` | `"Seyn"` |
| `"Daejun"` | `'a'` | `"Seyna"` |
| `"Daegu"` | `'a'` | `"Seynaa"` |
| `"Pusan"` | `'u'` | `"Seynaaу"` |

---

## 출력 결과

```
Seynaaу
```
