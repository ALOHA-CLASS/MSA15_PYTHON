# 2025년 1회 - 이진 트리 구성 & 재귀 계산

## 원본 코드

```python
class Node:
  def __init__(self, value):
    self.value = value
    self.children = []

def tree(li):
  nodes = [Node(i) for i in li]
  for i in range(1, len(li)):
    nodes[(i - 1) // 2].children.append(nodes[i])
  return nodes[0]

def calc(node, level=0):
  if node is None:
    return 0
  return (node.value if level % 2 == 1 else 0) + sum(calc(n, level + 1) for n in node.children)

li = [3, 5, 8, 12, 15, 18, 21]
root = tree(li)
print(calc(root))
```

---

## 문법 해설

### 클래스 (`Node`)

```python
class Node:
  def __init__(self, value):
    self.value = value      # 노드 값
    self.children = []      # 자식 노드 리스트
```

- `__init__` : 인스턴스 생성 시 자동 호출되는 생성자
- `self` : 인스턴스 자신을 가리키는 참조

---

### 리스트 컴프리헨션

```python
nodes = [Node(i) for i in li]
# li의 각 값으로 Node 객체 생성 → 리스트에 저장
# [Node(3), Node(5), Node(8), Node(12), Node(15), Node(18), Node(21)]
```

---

### 완전 이진 트리 구성 원리

```python
for i in range(1, len(li)):
    nodes[(i - 1) // 2].children.append(nodes[i])
```

- 인덱스 `i`의 부모 인덱스 = `(i-1) // 2`

| i (자식) | (i-1)//2 (부모) |
|-----------|-----------------|
| 1 | 0 |
| 2 | 0 |
| 3 | 1 |
| 4 | 1 |
| 5 | 2 |
| 6 | 2 |

---

### 트리 구조 시각화

```
          Node(3)      ← level 0
         /       \
     Node(5)   Node(8)   ← level 1
     /   \     /   \
  Node(12) Node(15) Node(18) Node(21)  ← level 2
```

---

### 재귀 함수 `calc`

```python
def calc(node, level=0):
  return (node.value if level % 2 == 1 else 0) + \
         sum(calc(n, level + 1) for n in node.children)
```

- `level % 2 == 1` (홀수 레벨)이면 `node.value` 포함, 짝수 레벨이면 `0`
- 자식 노드들에 대해 재귀 호출 후 합산

---

### 삼항 연산자 (Conditional Expression)

```python
node.value if level % 2 == 1 else 0
# level이 홀수면 node.value, 짝수면 0
```

---

### calc 실행 흐름

| 노드 | level | level%2 | 포함 여부 | 값 |
|------|-------|---------|-----------|-----|
| Node(3) | 0 | 0 (짝수) | ❌ | 0 |
| Node(5) | 1 | 1 (홀수) | ✅ | 5 |
| Node(8) | 1 | 1 (홀수) | ✅ | 8 |
| Node(12) | 2 | 0 (짝수) | ❌ | 0 |
| Node(15) | 2 | 0 (짝수) | ❌ | 0 |
| Node(18) | 2 | 0 (짝수) | ❌ | 0 |
| Node(21) | 2 | 0 (짝수) | ❌ | 0 |

합계: `0 + 5 + 8 + 0 + 0 + 0 + 0` = **13**
