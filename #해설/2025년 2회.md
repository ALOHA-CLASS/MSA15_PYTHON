# 2025년 2회 - `enumerate()` & 딕셔너리 & 튜플

## 원본 코드

```python
data = [
  [3, 5, 2, 4, 1],
  [4, 5, 1],
  [4, 4, 1, 5, 4],
  [4, 5]
]

result = {}

for index, lis in enumerate(data):
  list_sum = sum(lis)
  list_len = len(lis)
  result[index] = (list_sum, list_len)

print(result)
```

---

## 문법 해설

### `enumerate()` — 인덱스 + 값 동시 순회

```python
for index, lis in enumerate(data):
```

- `enumerate(iterable)` : `(인덱스, 값)` 튜플 쌍을 반환하는 이터레이터
- 언패킹으로 `index`와 `lis`에 동시 할당

```python
# enumerate 없이 쓸 경우
for i in range(len(data)):
    lis = data[i]
    index = i

# enumerate 사용 (더 파이썬스러운 방식)
for index, lis in enumerate(data):
    ...
```

---

### `sum()` & `len()` 내장 함수

```python
list_sum = sum(lis)   # 리스트 원소의 합
list_len = len(lis)   # 리스트 원소의 개수
```

---

### 딕셔너리에 값 저장

```python
result = {}              # 빈 딕셔너리 생성
result[index] = (list_sum, list_len)   # key: index, value: 튜플
```

- `result[key] = value` : 딕셔너리에 key-value 쌍 추가/수정

---

### 튜플 (Tuple)

```python
(list_sum, list_len)
```

- `()` 소괄호로 생성, 순서 있음, **수정 불가(immutable)**
- 두 값을 묶어 딕셔너리의 value로 저장

---

## 실행 흐름

| index | lis | sum | len | result[index] |
|-------|-----|-----|-----|---------------|

| 3 | [4,5] | 9 | 2 | (9, 2) |
