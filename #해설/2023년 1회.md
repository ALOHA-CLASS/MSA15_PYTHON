# 2023년 1회 - 세트(Set) 연산 (2020년 2회와 동일)

## 원본 코드

```python
a = {'한국', '중국', '일본'}
a.add('베트남')
a.add('중국')
a.remove('일본')
a.update({'홍콩', '한국', '태국'})
print(a)
```

---

## 문법 해설

### 세트(Set) 자료형

```python
a = {'한국', '중국', '일본'}
```

- `{}` 로 생성 (딕셔너리와 구분: 딕셔너리는 `{key: value}`)
- **특징**: 중복 불허, 순서 없음(unordered)

---

### 세트 주요 메서드

| 메서드 | 기능 | 중복 처리 |
|--------|------|-----------|
| `.add(값)` | 원소 1개 추가 | 중복 시 무시 |
| `.remove(값)` | 원소 1개 삭제 | 없으면 `KeyError` |
| `.discard(값)` | 원소 1개 삭제 | 없어도 오류 없음 |
| `.update(iterable)` | 여러 원소 추가 | 중복 시 무시 |

---

## 실행 흐름

| 단계 | 연산 | 세트 상태 |
|------|------|-----------|
| 초기 | | {한국, 중국, 일본} |
| `add('베트남')` | 추가 | {한국, 중국, 일본, 베트남} |
| `add('중국')` | 중복 → 무시 | {한국, 중국, 일본, 베트남} |
| `remove('일본')` | 삭제 | {한국, 중국, 베트남} |
| `update({'홍콩','한국','태국'})` | 홍콩·태국 추가, 한국 중복 무시 | {한국, 중국, 베트남, 홍콩, 태국} |

---

## 출력 결과

```
{'한국', '중국', '베트남', '홍콩', '태국'}
```

> ⚠️ 세트는 **순서가 없으므로** 실행마다 출력 순서가 다를 수 있음  
> 시험에서는 **포함된 원소 5개**가 맞는지 확인
