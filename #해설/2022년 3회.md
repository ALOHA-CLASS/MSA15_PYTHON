# 2022년 3회 - lambda & map

## 원본 코드

```python
TestList = [1, 2, 3, 4, 5]
TestList = list(map(lambda num: num + 100, TestList))
print(TestList)
```

---

## 출력 결과

```
[101, 102, 103, 104, 105]
```

---

## 문법 해설

### lambda (람다 함수)

```python
lambda num: num + 100
```

- `def` 없이 **한 줄로** 함수를 만드는 방법
- `lambda 매개변수: 반환값` 형태
- 위 코드에서는 `num`을 받아 `num + 100`을 반환

---

### map()

```python
map(lambda num: num + 100, TestList)
```

- `map(함수, 반복가능객체)` : 리스트의 **각 요소에 함수를 적용**
- 결과는 `map 객체`로 반환 → `list()`로 변환 필요

---

### 처리 흐름

| 원소 | 처리 (`num + 100`) | 결과 |
|------|-------------------|------|
| 1 | 1 + 100 | 101 |
| 2 | 2 + 100 | 102 |
| 3 | 3 + 100 | 103 |
| 4 | 4 + 100 | 104 |
| 5 | 5 + 100 | 105 |

---

### list() 변환

```python
TestList = list(map(...))
```

- `map()`의 결과는 즉시 리스트가 아님
- `list()`로 감싸야 **리스트로 변환**됨
