## 원본 코드

```python
lol = [[1,2,3],[4,5],[6,7,8,9]]
print(lol[0])
print(lol[2][1])
for sub in lol:
  for item in sub:
    print(item, end = '')
  print()
```

---

## 문법 해설

### 중첩 리스트 (List of Lists)

```python
lol = [[1,2,3],[4,5],[6,7,8,9]]
```

- 리스트 안에 리스트를 원소로 갖는 2차원 구조
- `lol[0]` → 첫 번째 원소인 리스트 `[1, 2, 3]`
- `lol[2][1]` → 세 번째 리스트 `[6,7,8,9]` 의 인덱스 1번 → `7`

---

### 인덱싱

```python
print(lol[0])      # [1, 2, 3]     — 서브 리스트 전체 출력
print(lol[2][1])   # 7             — lol[2]=[6,7,8,9], [6,7,8,9][1]=7
```

| 표현 | 값 |
|------|----|
| `lol[0]` | `[1, 2, 3]` |
| `lol[1]` | `[4, 5]` |
| `lol[2]` | `[6, 7, 8, 9]` |
| `lol[2][0]` | `6` |
| `lol[2][1]` | `7` |

---

### 중첩 for문 + `end` 파라미터

```python
for sub in lol:          # sub = 각 서브 리스트
    for item in sub:     # item = 서브 리스트의 각 원소
        print(item, end='')  # 줄바꿈 없이 출력
    print()              # 서브 리스트 하나 끝날 때마다 줄바꿈
```

- `print(item, end='')` : 기본 `end='\n'`(줄바꿈) 대신 빈 문자열 → 같은 줄에 이어 출력
- 내부 루프 종료 후 `print()` → 줄바꿈만 출력

---

## 실행 흐름

```
sub = [1,2,3]  → 1 출력, 2 출력, 3 출력, 줄바꿈
sub = [4,5]    → 4 출력, 5 출력, 줄바꿈
sub = [6,7,8,9]→ 6 출력, 7 출력, 8 출력, 9 출력, 줄바꿈
```
